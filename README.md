# CountingPrototype

## Challenge Outcome

- The prototype continues to count a chosen element during gameplay  
- A UI component is displayed that shows the current count to the player  
- The design document explains the functionality of the prototype based on the supplied base project  
- The Unity prototype implements the functionality described in the design document
